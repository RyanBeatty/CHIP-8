A CHIP-8 emulator/interpreter written in C

# Dependencies
SDL 2
```
sudo apt-get install libsdl2-2.0
sudo apt-get install libsdl2-dev
```

# Build

```
make all
```

# Run

```
./chip8 path/to/rom
```

Keys are mapped as follows:
```

Chip-8       QWERTY
-------      -------      
1|2|3|C      1|2|3|4
-------      -------
4|5|6|D      Q|W|E|R
-------      -------
7|8|9|E      A|S|D|F
-------      -------
A|0|B|F      Z|X|C|V
-------      -------
```
